Methods

seek()

seek(  $to = self::TO_NEXT) : boolean

Moves scope cursor TO_START, TO_END or TO_NEXT.

Scope cursor moves over array of tuples stored in the array scope->tuplesByNo[]. This scope->curTuple is an active tuple and scope->curTupleKey contains current tuple key value that could be used for accessing tuple by key value. Tuples arranged by keys are stored in scope->tuplesByKey[]

Parameters

$to

Returns

boolean —

true if end of index is reached
